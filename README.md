# プロジェクト名（例：販売管理システム）

Delphi 5 で作成した、顧客情報と売上を管理するツールです。

## 1. 動作環境
* OS: Windows 10 / 11 (32bit/64bit)
* 開発環境: Delphi 5 Enterprise (Service Pack 1)
* データベース: BDE / SQLServer

## 2. 使い方
1. `SalesManager.dpr` を Delphi で開きます。
2. `F9` キーでコンパイルおよび実行します。
3. 実行ファイルはプロジェクトと同じフォルダに生成されます。

## 3. 注意点
* **BDEのインストールが必要です。**
* 実行前に `data` フォルダ内のパス設定を確認してください。
